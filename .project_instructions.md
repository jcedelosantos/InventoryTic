# Inventario TIC - Reverse

## Descripción
Aplicación de gestión de inventario tecnológico y auditoría para el cliente Reverse. Incluye módulos para equipos, licencias, consumos mensuales, soportes tercerizados y proyectos en curso.

## Stack Técnico
- App Router, Prisma ORM, PostgreSQL
- Tailwind CSS, Framer Motion, Recharts
- NextAuth.js v4 (email/password)
- xlsx para importación/exportación Excel

## Paleta de Colores
- Primary: Azul profesional (#1E40AF / #3B82F6)
- Accent: Teal (#0D9488)
- Background: #F8FAFC (light gray-blue)
- Cards: White with subtle shadows
- Danger: #EF4444
- Warning: #F59E0B
- Success: #10B981

## Idioma
- Toda la interfaz en español
- Moneda: USD con formato $X,XXX.XX

## Módulos
1. Dashboard - Resumen ejecutivo
2. Equipos Tecnológicos - CRUD + importación Excel
3. Licencias Anuales - 13 categorías
4. Consumos Mensuales - 3 categorías
5. Soportes Tercerizados - 3 proveedores
6. Proyectos en Curso - 4 proyectos

## Decisiones de Diseño
- Sidebar navigation con iconos lucide-react
- Tablas con paginación y búsqueda
- Formularios con modales
- Gráficos recharts para dashboard
