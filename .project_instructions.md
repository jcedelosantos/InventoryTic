# Inventario TIC - Cedanet Solutions

## Descripción
Aplicación multi-cliente de gestión de inventario tecnológico y auditoría para Cedanet Solutions. Permite gestionar equipos, licencias, consumos, soportes y proyectos de múltiples clientes desde una sola interfaz.

## Empresa
- **Cedanet Solutions** - Empresa de servicios IT
- Primer cliente: **Reverse** (salón/belleza)

## Stack Técnico
- App Router, Prisma ORM, PostgreSQL
- Tailwind CSS, Framer Motion, Recharts
- NextAuth.js v4 (email/password)
- xlsx para importación/exportación Excel

## Arquitectura Multi-Cliente
- Modelo `Client` en DB con relaciones a todos los modelos de datos
- `ClientProvider` (contexts/client-context.tsx) con selector en sidebar
- Todas las APIs filtran por `clientId` query param
- Exportación/importación filtrada por cliente
- ClientProvider usa `useSession` para evitar fetch antes de autenticación

## Idioma
- Toda la interfaz en español
- Moneda: USD con formato $X,XXX.XX

## Módulos
1. Dashboard - Resumen ejecutivo (filtrado por cliente)
2. Equipos Tecnológicos - CRUD + importación Excel + campo Serial/SN
3. Licencias Anuales
4. Consumos Mensuales
5. Soportes Tercerizados
6. Proyectos en Curso
7. Clientes - Gestión CRUD de clientes
